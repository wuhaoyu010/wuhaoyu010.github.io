安装gulp

我推荐使用最新版的gulp 4.0，为了更方便的使用gulp的顺序执行任务功能。此处知识在这里了解到的。

我安装第一个的时候失败了，但不影响使用的。

安装gulp插件

在这里需要以下：

  • gulp-minify-css 压缩css
  • gulp-htmlmin 压缩html
  • gulp-htmlclean html清理
  • gulp-uglify 压缩js
  • gulp-imagemin 压缩图片
  • gulp-clean 清理文件或目录
  • gulp-shell

安装命令

npm install [plugins-name] --save

plugins-name在上面呦
